# penguin-signpost 

wassan128のブログです

## 使い方
記事ファイルの作成には `penguin-signpost` コマンド(シェルスクリプト)を使用します。以下コマンドを実行すると記事のテンプレートファイルを生成します。

```
$ git clone $THIS_REPOSITORY
$ cd penguin-signpost
$ penguin-signpost create
[*] 記事を作成します...

[*] 記事ファイルを作成しました: docs/archives/2020/08/05/README.md
```
